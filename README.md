# Kanban Task Board

A Kanban task board built with HTML, CSS, and JavaScript.

## Live Demo

https://jsantos1387.github.io/kanban-task-board/

## Features

- Add tasks
- Delete tasks
- Move tasks between columns
- Drag and drop functionality
- Task priorities (Low, Medium, High)
- Due dates
- Task counters
- Data saved with localStorage
- Automatically loads saved tasks after refresh

## Technologies Used

- HTML
- CSS
- JavaScript
- GitHub Pages
